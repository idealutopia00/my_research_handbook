# Smart Mining Theme

This directory contains chapters related to smart mining, digital twins, parallel systems, and intelligent mining technologies.

## Current Content
- `2025_2026_winterbreak/` - Winter break research on smart mining development

## Adding New Content

### Using the Script
```bash
python ../../../scripts/new_chapter.py "digital_twin_applications" --theme "smart_mining"
```

### Manual Method
1. Create a new directory (e.g., `digital_twin_advances/`)
2. Copy `../../../settings/template.tex` to `main.tex`
3. Edit the content
4. Update `../../../../main.tex` to include the new chapter
