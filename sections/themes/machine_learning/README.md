# Machine Learning Theme

This directory contains chapters related to machine learning algorithms, deep learning, transformers, and related methodologies.

## Example Topics
- Deep learning architectures
- Transformer models
- Reinforcement learning
- Federated learning
- Explainable AI

## Adding New Content

### Using the Script
```bash
python ../../../scripts/new_chapter.py "transformer_architectures" --theme "machine_learning"
```

### Manual Method
1. Create a new directory with descriptive name
2. Add `main.tex` with chapter content
3. Update main document to include the chapter

## Bibliography
References are in `bib/uncategorized.bib`. Consider creating `bib/machine_learning.bib` for better organization.
